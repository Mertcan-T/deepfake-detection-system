# Deepfake Tespit Projesi

Videolarda deepfake manipulasyonlarını derin öğrenme kullanarak tespit eden bir Python projesi.

## Proje Özeti

Bu proje, üretken yapay zeka (Generative AI) teknolojileri kullanılarak oluşturulan sahte videoları tespit etmeyi amaçlamaktadır. Projede CNN tabanlı derin öğrenme modelleri, video işleme ve biyometrik analiz teknikleri kullanılmaktadır.

## Amaçlar

- Gerçek videolardan deepfake videolarını ayırt edebilen model geliştirmek
- Piksel seviyesinde yapay üretime özgü anomalileri tespit etmek  
- Biyometrik veriler (göz kırpışı, yüz hareketi) aracılığıyla sahte içeriği belirlemek
- Yüksek doğrulukta (%90+) bir sınıflandırıcı model eğitmek
